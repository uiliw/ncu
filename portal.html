Este é um programa fechado para as universidades. Para realizar o Programa Negócio Certo para Universitários você precisa:

  • Estar cursando graduação.
  • Sua universidade deve estar conveniada com este programa.

Se neste momento você não está cursando graduação ou a sua universidade não está conveniada, você poderá se cadastrar gratuitamente no Programa de Auto-Atendimento Negócio Certo acessando o seguinte endereço:
www.sebrae-sc.com.br/negociocerto
